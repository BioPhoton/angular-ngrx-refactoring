Types of state:

Server State

The server state is stored on the server and provided over a REST-API endpoint. (www.angular.at/api)

Persistent State

The persistent state is a subset of the server state stored on the client. It can be stored in memory in the local stored or anywhere else.

Router/URL state

The router/URL state is present in the browser search bar

Client state

The values in the search form

Local UI state

The local UI state describes some component internal UI state. i.e. a progress spinner
